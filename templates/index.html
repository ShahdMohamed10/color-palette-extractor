API Information

To use the color extraction API, send a POST request to /api/extract-colors with:

  • Form parameter image: The image file
  • Form parameter num_colors: Number of colors to extract (optional, default is 5)

Example curl command:

curl -X POST -F "image=@path/to/your/image.jpg" -F "num_colors=5" https://your-site.pythonanywhere.com/api/extract-colors
